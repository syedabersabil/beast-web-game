# Beast Web Game

An action-packed beast hunting game built with React.js and Canvas API.

## Features

- **Beast Hunting**: Shoot down waves of attacking beasts
- **Progressive Difficulty**: Each wave spawns more beasts
- **Health System**: Player health decreases on collision with beasts
- **Score System**: Earn points for destroying beasts
- **Wave Progression**: Complete waves to advance to harder challenges
- **Smooth Controls**: WASD movement with mouse click to shoot
- **Visual Feedback**: Health bars for both player and beasts

## Controls

- **W/A/S/D** or **Arrow Keys** - Move around
- **Left Click** - Shoot projectiles

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
